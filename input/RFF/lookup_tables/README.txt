Lookup tables contain age specific ratios for each country of the population and respiratory mortality projections in each RFF scneario relative to the average across all scenarios.

Ratios are used as input into the Reducted Form Tool

Lookup tables are generated using the 'Create RFF lookup Tables.R' script. 

Ouptu data are too large for github file size limits. Contact mcduffie.erin.e@epa.gov for further information.
